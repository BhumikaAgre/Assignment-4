Task 1: Read a File and Handle Errors

 Description
This program:
- Opens and reads a text file named `sample.txt`
- Prints the file content line by line
- Handles errors gracefully if the file does not exist

Concepts Used
- File reading (`open`, `read`)
- Exception handling (`try-except`)
- Line numbering using `enumerate()`

Expected Output
- If the file exists, it displays the content line by line
- If the file does not exist, it shows an error message

  Task 2: Write and Append Data to a File

Description
This program:
- Takes user input and writes it to `output.txt`
- Appends additional user input to the same file
- Reads and displays the final content of the file

Concepts Used
- File write mode (`w`)
- File append mode (`a`)
- File read mode (`r`)
- User input handling

Expected Output
- Confirms successful write and append operations
- Displays the final content of `output.txt`
